# Web 测试与验收清单 v0.1

本文档记录 `note-remeber` Web 端第一版的测试与验收标准。

本文档基于：

```text
01-web-product-spec.md
03-web-backend-design.md
04-web-frontend-design.md
05-supabase-database-sql-design.md
06-web-data-api-design.md
07-rich-editor-and-flowchart-design.md
08-deployment-and-auth-setup.md
```

目标是确保第一版开发完成后，不只是“能打开页面”，而是真的满足产品核心逻辑。

## 1. 验收范围

第一版验收范围包括：

```text
登录与权限
问题库
回答标题页
正文编辑
问题见解文
文章见解文
衍生问题
衍生跳转
顶部路径
流程图
图片上传
搜索与防重名
删除确认与级联清理
Supabase RLS
Netlify 部署
响应式体验
错误状态
```

## 2. 验收环境

至少需要验证两个环境：

```text
本地开发环境：http://localhost:3000
Netlify 生产或预览环境：https://<site>.netlify.app
```

本地验证命令：

```text
npm run dev
npm run build
```

生产验证：

```text
Netlify build 成功
线上页面可访问
线上登录可用
线上数据读写可用
```

## 3. 登录与权限验收

### 3.1 未登录状态

必须满足：

```text
未登录用户进入站点时显示登录页
未登录用户不能看到问题库数据
未登录用户不能通过 URL 直接打开正文数据
未登录用户不能调用业务写入操作
```

### 3.2 邮箱 OTP 登录

必须满足：

```text
输入已授权邮箱可以发送验证码
输入验证码后可以登录
登录成功后进入问题库首页
验证码错误时显示明确提示
验证码过期时显示明确提示
未授权邮箱不能自动创建账号
```

### 3.3 退出登录

必须满足：

```text
可以退出登录
退出后回到登录页
退出后不能继续看到缓存中的业务数据
```

### 3.4 RLS 权限

必须满足：

```text
用户只能读取自己的 questions
用户只能读取自己的 answer_articles
用户只能读取自己的 documents
用户只能读取自己的 flowcharts
用户只能读取自己的 assets
用户不能把自己的记录关联到别人的父级记录
用户不能访问别人 Storage 路径下的文件
```

## 4. 问题库验收

### 4.1 问题列表

必须满足：

```text
登录后显示问题库首页
问题列表按更新时间或指定排序显示
每个问题显示标题
每个问题显示回答数量
每个问题显示非空正文数量
无回答问题显示为无回答状态
有标题但无正文问题显示为仅标题状态
有非空正文问题显示为有正文状态
```

### 4.2 新建问题

必须满足：

```text
点击新建问题可以输入标题
输入时实时显示相似问题
完全重名问题不能创建
标题为空不能创建
相似但不完全相同的问题允许继续创建
创建成功后问题出现在问题库
创建成功后可以进入该问题的回答标题页
```

### 4.3 搜索问题

必须满足：

```text
搜索“发”能返回包含“发”的问题
搜索“发动机”能返回包含“发动机”的问题
搜索完整标题时能收窄到目标问题
清空搜索后恢复完整列表
```

### 4.4 编辑问题标题

必须满足：

```text
问题标题可以编辑
编辑后自动保存
编辑为空时阻止保存
编辑成完全重名时阻止保存
保存成功后列表和路径同步更新
```

### 4.5 删除问题

必须满足：

```text
删除前弹出确认框
点击取消不会删除
点击确认才会删除
删除后问题从问题库消失
删除后该问题下回答标题被清理
删除后该问题下正文被清理
删除后该问题下流程图记录被清理
删除后该问题下 assets 记录被清理
删除不会误删其他问题本体
```

## 5. 回答标题页验收

### 5.1 页面展示

必须满足：

```text
点击问题后进入回答标题页
页面显示当前问题标题
页面显示回答数量和非空正文数量
页面显示问题见解入口
页面显示回答文章列表
```

### 5.2 新建回答标题

必须满足：

```text
可以在问题下新建回答标题
输入时实时显示相似标题
同一问题下完全重名标题不能创建
不同问题下可以存在相同标题
创建成功后生成空 article_body
创建成功后标题出现在列表
```

### 5.3 回答标题搜索

必须满足：

```text
标题搜索采用字查询
搜索结果只限当前问题下的回答标题
清空搜索后恢复当前问题所有标题
```

### 5.4 编辑回答标题和来源

必须满足：

```text
回答标题可以编辑
编辑成同问题下完全重名时阻止保存
来源链接可以编辑
来源链接为空时允许保存
```

### 5.5 首选回答

必须满足：

```text
可以设置某篇回答为首选回答
同一个问题最多只有一个首选回答
设置新的首选回答后旧首选自动取消
首选回答标记在列表中清晰可见
```

### 5.6 删除回答标题

必须满足：

```text
删除前弹出确认框
取消不会删除
确认后删除标题
确认后删除对应文章原文正文
确认后删除对应文章见解文
确认后删除对应流程图记录
确认后删除对应衍生问题引用
确认后不删除衍生问题本体
```

## 6. 正文编辑验收

### 6.1 文章原文正文

必须满足：

```text
点击回答标题进入文章原文正文
正文为空时显示 placeholder
可以输入正文
可以粘贴文章
刷新后正文内容仍存在
正文保存状态显示准确
```

### 6.2 问题见解文

必须满足：

```text
回答标题页可以打开问题见解文
问题见解文为空时允许进入
问题见解文可以编辑和保存
问题见解文可以显示衍生问题栏
问题见解文可以打开流程图
问题见解文可以跳转到首选回答文章
```

### 6.3 文章见解文

必须满足：

```text
文章正文页可以打开文章见解文
文章见解文可以编辑和保存
文章见解文可以打开流程图
文章见解文不显示衍生问题栏
文章见解文可以返回文章原文正文
```

### 6.4 富文本格式

必须满足：

```text
加粗可用
斜体可用
下划线可用
删除线可用
标题级别可用
有序列表可用
无序列表可用
引用可用
链接可插入和编辑
文字颜色可用
背景高亮可用
表格可插入和编辑
分割线可插入
清除格式可用
```

### 6.5 粘贴保留格式

必须满足：

```text
从网页复制文章后能保留基础段落结构
从网页复制文章后能保留标题
从网页复制文章后能保留加粗、链接、列表
从 Word 复制内容后能保留主要结构
粘贴内容不会带入 script
粘贴内容不会执行危险事件属性
```

### 6.6 自动保存

必须满足：

```text
输入后标记为未保存
停止输入后自动保存
保存中显示 saving
保存成功显示 saved
保存失败显示 error
保存失败不丢失编辑器当前内容
切换页面前尽量保存未提交内容
```

## 7. 图片与资源验收

必须满足：

```text
可以上传本地图片
可以粘贴截图图片
图片上传到 Supabase Storage
图片生成 assets 记录
正文中显示上传后的图片
刷新后图片仍能显示
图片路径以 user_id 开头
删除正文时 assets 记录被清理
问题或文章删除时 Storage 文件尽量被清理
上传失败时显示重试或错误提示
```

暂不强制第一版完成：

```text
外链图片自动转存
图片说明文字
图片压缩
```

## 8. 衍生问题验收

### 8.1 衍生问题栏显示

必须满足：

```text
文章原文正文显示衍生问题栏
问题见解文显示衍生问题栏
文章见解文不显示衍生问题栏
衍生跳转模式下不显示衍生问题栏
衍生问题显示标题
衍生问题显示回答数量
衍生问题显示非空正文数量
衍生问题显示见解状态
```

### 8.2 新建衍生问题

必须满足：

```text
新建衍生问题时实时搜索全局问题库
当前正文已有该衍生问题时不能重复添加
全局问题库已有该问题时复用已有问题
全局问题库没有该问题时创建新问题
复用已有问题后立即显示已有回答数和正文数
新建后该问题出现在问题库首页
```

### 8.3 从选中文本创建衍生问题

如果第一版实现该能力，则必须满足：

```text
选中文字后出现新建衍生问题入口
默认问题标题可基于选中文本生成
文章见解文中不显示该入口
衍生跳转模式中不显示该入口
```

### 8.4 删除衍生问题引用

必须满足：

```text
删除前弹出确认框
确认后只删除当前正文与该问题的关联
不会删除问题库里的问题
不会影响其他正文里的同一衍生问题
```

## 9. 衍生跳转验收

### 9.1 跳转到问题见解

必须满足：

```text
点击衍生问题可以选择查看问题见解
问题见解存在时进入该见解文
问题见解不存在时自动创建空见解文并进入
空见解文显示暂无内容提示
问题见解文中提供跳转首选回答按钮
```

### 9.2 跳转到首选回答

必须满足：

```text
点击衍生问题可以选择查看首选回答
有首选回答且正文非空时跳转首选回答正文
没有首选回答但有其他非空正文时跳转第一个有正文的回答
只有标题但无非空正文时显示无法跳转
没有回答标题时显示无法跳转
```

### 9.3 跳转深度限制

必须满足：

```text
从原正文可以进行一次衍生跳转
衍生跳转后的目标页不显示衍生问题栏
不能继续从目标页跳到第三个问题
同一目标问题内可以在见解文和首选回答之间切换
同一目标问题内切换不算第二次衍生跳转
```

## 10. 顶部导航路径验收

必须满足：

```text
问题库首页不显示复杂路径
回答标题页显示首页 > 标题页
文章正文页显示首页 > 标题页 > 正文页
衍生跳转后路径追加衍生跳转项
路径中的每一项可以点击返回
问题库按钮回到首页并清空路径
回答按钮回到当前原始问题的回答标题页
从衍生跳转页可以返回原正文
返回原正文时尽量恢复滚动位置
```

```text
文章原文正文可以打开流程图面板
文章见解文可以打开流程图面板
问题见解文可以打开流程图面板
不同正文的流程图互相独立
```

### 11.2 流程图编辑

必须满足：

```text
可以新建流程图
可以编辑流程图标题
可以新增节点
可以编辑节点文字
可以拖拽节点
可以创建连线
可以删除节点
可以删除连线
可以缩放画布
可以拖动画布
可以全屏编辑
可以退出全屏
```

### 11.3 流程图保存

必须满足：

```text
流程图编辑后自动保存
保存状态显示准确
刷新后节点仍存在
刷新后连线仍存在
刷新后画布状态尽量保留
```

必须满足：

```text
删除流程图前弹出确认框
```

## 12. 搜索与防重名验收

必须满足：

```text
问题库搜索采用字查询
回答标题搜索采用字查询
衍生问题栏搜索采用字查询
新建问题时显示相似问题
新建标题时显示相似标题
新建衍生问题时显示相似全局问题
完全重名问题由数据库阻止
同问题下完全重名标题由数据库阻止
当前正文重复衍生问题由数据库阻止
数据库错误转换为用户可读提示
```

## 13. 删除与清理验收

必须满足：

```text
所有删除操作都有确认框
确认框有取消和确认按钮
危险确认按钮视觉上明确
取消不会产生任何删除
删除问题会清理问题下所有回答和正文数据
删除文章会清理文章正文和见解文数据
删除衍生问题只删除关联
删除流程图只删除流程图
Storage 清理失败时有 warning 或可追踪信息
```

禁止出现：

```text
未确认直接删除
删除衍生问题时误删问题库问题
删除文章时误删其他文章
删除问题时误删其他独立问题
```

```text
衍生问题栏可以作为抽屉打开

按钮文字不溢出
```

## 18. 最小完整用户路径

第一版至少要跑通以下完整路径：

```text
1. 登录
2. 创建问题 A
3. 进入问题 A 的回答标题页
4. 创建回答文章 A1
5. 进入 A1 原文正文
6. 粘贴一篇文章
7. 保存正文
8. 在 A1 中创建衍生问题 B
9. 回到问题库，看到问题 B
10. 给问题 B 写问题见解 B+
11. 回到 A1，点击衍生问题 B
12. 跳转到 B+ 问题见解
13. 返回 A1 原文正文
14. 给 A1 写文章见解 A1+
15. 在 A1 或 A1+ 中创建流程图
16. 设置 A1 为问题 A 的首选回答
17. 删除一个衍生问题引用，确认问题 B 本体仍存在
18. 删除 A1，确认问题 B 本体仍存在
```


## 20. 当前验收总结

第一版验收通过的标准不是页面做完，而是以下能力稳定：

```text
问题能长期保存
正文能可靠编辑和保存
衍生问题能连接问题网络
跳转不会让用户迷路
流程图能辅助理解
删除不会误伤数据
权限不会泄露数据
线上环境能真实使用
```
